# Esther Cui｜商业分析师作品集

面向商业分析、经营分析和数据分析岗位的个人作品集网站。

正式网站：[https://www.esthercui.cn](https://www.esthercui.cn)

## 技术栈

- React + TypeScript + Vite
- Tailwind CSS
- Framer Motion

## 本地运行

请先安装 Node.js，然后在项目目录中运行：

```bash
npm install
npm run dev
```

浏览器访问 `http://127.0.0.1:3000/`。

## 生产构建

```bash
npm run build
```

## 内容维护

- 项目内容：`src/data/projects.ts`
- 技能内容：`src/data/skills.ts`
- 页面组件：`src/components/`
- 图片资源：`public/`
